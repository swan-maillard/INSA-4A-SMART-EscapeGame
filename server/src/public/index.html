API Documentation

[FOR DEV ONLY] Users

GET /users

Send all users.

Server Response:

  • 200 OK Returns an array of user objects.

GET /users/:id

Send user with given ID.

Server Response:

  • 200 OK Returns the user object with the given ID.
  • 404 Not Found If the user with the given ID does not exist.

Games

GET /games

[FOR DEV ONLY] Send all game sessions.

Server Response:

  • 200 OK Returns an array of game objects.

GET /games/:id

Send game session with given ID.

Server Response:

  • 200 OK Returns the game object with the given ID.
  • 401 Unauthorized You are not authorized because you don't play in this game.
  • 404 Not Found If the game with the given ID does not exist.

POST /games/create

Create a game session.

Parameters:

  • username The username of the creator.

Server Response:

  • 200 OK Returns the created game object and the Authorization Token (token).
  • 400 Bad Request If the request parameters are invalid.

POST /games/join

Join a game session.

Parameters:

  • username The username of the player.
  • gameId The ID of the game session to join.

Server Response:

  • 200 OK Returns the updated game object and the Authorization Token (token).
  • 400 Bad Request If the request parameters are invalid.
  • 404 Not Found If the game with the given ID does not exist.

POST /games/delete

Delete a game session.

Parameters:

  • gameId The ID of the game session to delete.

Server Response:

  • 200 OK Returns a success message.
  • 400 Bad Request If the request parameters are invalid.
  • 404 Not Found If the game with the given ID does not exist.
